Language Version : ActionScript 3.0
Runtime Versions : AIR 2.0, Flash Player 10

Use BitmapData as source and implement grid access to bitmap parts ( cell system ).

Allows to build a bitmap with many icons, or anything else inside and access specific one using grid access.
