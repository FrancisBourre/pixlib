firePrivateEvent()method 
public static function firePrivateEvent(event:Event):void

Language Version : ActionScript 3.0
Runtime Versions : AIR 2.0, Flash Player 10

Fires passed-in event in private event communication channel.

Passed-in event type must be registered into BasePlugin FrontController to be dispatched. Used to build an Event type/Command association.

Parameters

event:Event — Event to dispatch in private communication channel


Example 
         
         public function registerCommand() : void
         {
             PXQuick.controller.pushCommandClass("onTest", TestCommand);
         }
         
         public function test() : void
         {
             PXQuick.firePrivateEvent(new Event("onTest"));
         }
