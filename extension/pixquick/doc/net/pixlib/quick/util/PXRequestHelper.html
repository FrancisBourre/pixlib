fromConfiguration()method 
public static function fromConfiguration(id:String, context:LoaderContext = null, anticache:Boolean = false):PXRequestHelper

Language Version : ActionScript 3.0
Runtime Versions : AIR 2.0, Flash Player 10

Creates and returns new PXRequestHelper instance using passed-in parameters.

Uses this method when you define differents url value in XML configuration file. Each node which define url must have an 'id' attribute.

Parameters

id:String — XML node attribute identifier.
 
context:LoaderContext (default = null) — (optional) Anticache value to use. If not defined, set to false.
 
anticache:Boolean (default = false) — (optional) Loading context to use. If not defined, use the PXQuick.getLoaderContext() context.

Returns
PXRequestHelper

Example 
         
         <configuration>
             <node id="portfolio" url="http://blog.customactions.net/portfolio" />
         </configuration>
Examples
     
     var helper : PXRequestHelper = PXRequestHelper.create("http://...");
     
     var loader : PXLoader = new PXGraphicLoader(); 
     loader.addEventListener(PXLoaderEvent.onLoadInitEVENT, onLoadInitHandler);
     loader.load(helper.request, helper.context);
